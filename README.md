# ZMQ Server

Сервер для приема и визуализации данных с ZeroMQ и графическим интерфейсом на ImGui.

## Описание

Проект реализует сервер, который принимает JSON-данные через ZeroMQ, сохраняет их в файл и отображает в реальном времени через GUI на ImGui.

## Зависимости

- **ZeroMQ** - транспортный уровень для приема данных
- **SDL2** - создание окна, обработка ввода
- **OpenGL/GLEW** - рендеринг графики
- **Dear ImGui** - интерфейс Immediate Mode GUI
- **STL** - потоки, mutex'ы, файловый ввод/вывод
